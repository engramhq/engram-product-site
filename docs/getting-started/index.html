Getting Started

Install Node

You can download and install Node.js from their official website

Install @engramhq/cli

$ npm i -g @engramhq/cli

Sign Up

If you already have an account you can use 
$ eg login
$ eg signup

Deploy!

Update project to be whatever you would like your project to be named. Your code will be deployed to https://project-username.cloud.engramhq.xyz
$ eg deploy -n project

Preview

Copy or click on the returned link and see your live website. 
$ Deployed to https://project-username.cloud.engramhq.xyz in 208ms
