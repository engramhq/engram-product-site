Getting Started

Install Node

You can download and install Node.js from their official website

Install @engramhq/cli

npm i -g @engramhq/cli
You can now run: 
eg --help
to see all available commands and options

Sign Up

You will be prompted for a username, email, and password. The username will be used as a prefix for your preview URLs.

eg signup

Once complete, a session is automatically stored on your system so you can use engram without having to log in again.

Create a New Project

eg new html-demo

This command creates the html-demo folder and copies a simple html template inside of it (more template projects are on the way)

Deploy

eg deploy html-demo

Preview

A browser should automatically open with your newly deployed project

Deployed to https://preview-username.cloud.engramhq.xyz in 208ms
Only a single preview link can be active at a time on the free plan
